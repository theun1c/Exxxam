Во время экзамена мы будет работать с AvaloniaUI и если она до сих пор не скачена на компе - это можно сделать просто через консоль (powershell) https://avaloniaui.net/ 

``` bash
dotnet new install Avalonia.Templates
```

После этого в вашем IDE будут доступны шаблоны авалонии. Теперь создадим проект AvaloniaUI MVVM .NET 9 - одна из стабильных версий. 

ВАЖНО! Перед экзаменом создайте аккаунт в https://portal.avaloniaui.net/ поскольку для установки плагина и "автодополнений" вам понадобится аккаунт. 

![](../assets/Pasted%20image%2020260529021716.png)

Также в расширениях скачайте дополнительно плагин для Avalonia. Просто переходите в "Расширения" -> Пишете AvaloniaUI и самый первый плагин скачивайте. (Именно для него и нужен будет аккаунт)

![](../assets/Pasted%20image%2020260529023755.png)

После скачивания примерно так будет выглядеть окно с входом. (оно может появится после использования превью ПКМ по AXAML и Designer)
